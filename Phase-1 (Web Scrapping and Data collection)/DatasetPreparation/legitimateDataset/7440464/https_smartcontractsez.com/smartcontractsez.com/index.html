Frequency Asked Questions

How to get started?

Simple! Follow our official documentation to get started. It’s a simple process to get your account registered with us and create api key

What are plan and pricing to use api tool?

It’s free to get started with your first api key. However you’ll require 50 ADA to deposit to our wallet. It costs 1 ADA to perform your very first transaction

Can I trade NFT and Token at an exchange?

You can simply transfer your existing NFT and tokens to another wallet. 

What is the difference between Smart Contract and NFT?

Both are blockchain technologies. Smart contract is a digital contracts where NFT can be embedded into the contract

Why SCEZ?

Is SCEZ api easy to implement and test with?

For sure, our api are well documented, robust and secure. So every api request will be authorized and authenticated.
