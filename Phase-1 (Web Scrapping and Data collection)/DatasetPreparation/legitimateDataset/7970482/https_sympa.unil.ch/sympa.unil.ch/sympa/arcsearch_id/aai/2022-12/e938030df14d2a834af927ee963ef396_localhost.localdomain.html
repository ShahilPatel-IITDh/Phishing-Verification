Welcome

This service gives you access to your mailing lists on sympa@unil.ch. From this page, you can choose your membership options, unsubscribe or manage lists that you have been named owner of, etc.

Some lists are configured to be read-only from this site; this is indicated in the list's Description section. These lists are managed in the Group Management application of the administrative intranet; go to the following link to do so https://www.unil.ch/ci/fr/home/menuinst/catalogue-de-services/authentification-et-comptes/gestion-des-groupes.html.

Logging In

Many functions in Sympa require you to identify yourself to the system by logging in, using the login form in the top right menu.
